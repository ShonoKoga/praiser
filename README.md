# praiser
『React Native -JavaScriptによるiOS / Androidアプリ開発の実践』 のサンプル実装


```
$ yarn install
```

### for ios
```
$ cd ios/
$ pod install --repo-update
$ cd ..
```

```
$ yarn ios

// or

$ npx react-native run-ios
```

### for android

With Android emulator running.

```
$ yarn android

// or

$ npx react-native run-andoid
```
